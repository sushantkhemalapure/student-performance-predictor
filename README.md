# Student Performance Predictor (Small AI Project)

This project predicts whether a student will PASS or FAIL using Machine Learning.
This is  project to understand basic ML 

## Inputs
- Study hours per day
- Attendance percentage
- Previous exam score

## Output
- PASS or FAIL

## Technologies Used
- Python
- pandas
- scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python model.py

## Example
Enter study hours per day: 5  
Enter attendance percentage: 80  
Enter previous exam score: 55  

Prediction: PASS


This project predicts result on the basis of Study hours per day, Attendance percentage and
 Previous exam score
